v2: http://jannisleidel.com/missingdrawer/

This plugin provides Xcode-like project window interface without drawer.
Screenshot is here.
v2: http://www.flickr.com/photos/jannis/2264845788/
v1: http://www.flickr.com/photos/hetima/292226114/

Known bugs:
* TextMate seems not to load the plugin if you drag a folder to its icon and it's not already running. You'll then get the "old" drawer layout.
* The same applies to opening existing TextMate project (<code>*.tmproj</code>) while TextMate is not running.</li>
